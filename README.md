## ⚙️ **Configuración local**

### 1. Clonar el repositorio:
```bash
git clone https://github.com/evers23/evers23.github.io
cd evers23.github.io

2. Instalar dependencias:
npm install

3. Iniciar el servidor de desarrollo:
npm run dev
